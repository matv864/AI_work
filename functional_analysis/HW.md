Пусть $A: l^n_p \to l^n_p$, $Ax=\{ \sum^{n}_{j=1}{a_{kj}\epsilon_j} \}^{n}_{k=1}$
Тогда  A - сжимающее отображение в пространстве $l^n_p$, если $$max_{1\le k\le n} \sum^{n}_{j=1}{|a_{kj}|} \lt 1 \,\, \text{при} \,\, p=\infty$$
норма $l_p$
$$||x||_{l_p}=(\sum^{\infty}_{1} |x_i|^p)^{1/p}, \,\, p \ge 1$$
при $p = \infty$
$$||x||_{l_{\infty}} = \max_i(x_i)$$

Оператор A сжимает, если существует $q < 1$, такое что для всех $x,y \in l^{n}_{\infty}$
$||Ax-Ay||_{\infty} \le q||x-y||_{\infty}$
$Ax-Ay = A(x-y)$
$] \, z = x-y$
$(Ax-Ay)_k = (Az)_k = \sum^{n}_{j=1}{a_{kj}z_j}$
$(Az)_k = |\sum^{n}_{j=1}{a_{kj}z_j}| \le \sum^{n}_{j=1}{|a_{kj}||z_j|}$  по неравенству треугольника

$|z_j| \le ||z||_{\infty}$
$\sum^{n}_{j=1}{|a_{kj}||z_j|} \le \sum^{n}_{j=1}{|a_{kj}| \, ||z||_{\infty}}$
$||Az||_{\infty} = max_{1\le k \le n}|(Az)_k| \le max_{1\le k \le n} \sum^{n}_{j=1}{|a_{kj}| \, ||z||_{\infty}}$


$] \, q = max_{1\le k \le n} \sum^{n}_{j=1}{|a_{kj}|} \,\,\, < 1$   из условия
$||Az||_{\infty} \le max_{1\le k \le n} \sum^{n}_{j=1}{|a_{kj}| \, ||z||_{\infty}}$
$||Az||_{\infty} \le q||z||_{\infty}$
$||Ax-Ay||_{\infty} \le q||(x-y)||_{\infty}$


### 10.6 вычислить норму функционала
$$f(x)=ax(0)+b\int^{1}_{0}x(t)dt, X=C[0,1]$$
$||f|| = \sup_{||x|| \le 1}{|f(x)|}$, 
где $||x|| = \max_{t \in [0,1]}{|x(t)|}$

$|f(x)| \le |a| \cdot |x(0)| + |b| \cdot |\int^{1}_{0}x(t)dt|$
Для любой $x \in C[0,1]$ с $||x|| \le 1$
$\to$ $|x(0)| \le 1$ и $|\int^1_0{x(t)dt}| \le \int^1_0{|x(t)|dt} \le 1$
$\to$ $|a \cdot x(0)| \le |a|$ и $|b \cdot \int^1_0{x(t)dt}| \le |b|$
$\to$ $|f(x)| \le |a| + |b|$

пусть $x(t) = sign(a)$ для всех $t$
тогда:
$x(0) = sign(a)$
$\int^1_0 x(t)dt = sign(a)$
представляем:
$f(x) = a \cdot sign(a) + b \cdot sign(a) = |a| + |b|$
$f(x) = |a|+|b|$
$||f(x)|| = |a|+|b|$


### 8.30 Пусть $H_0$ - одномерное подпространство в гильбертовом пространстве $H$, $x_0 \in H_0$, $x_0 \ne 0$. Доказать, что для любого $x \in H$
$$p(x, H^{\bot}_0) = \frac{|(x, x_0)|}{||x_0||}$$

$p(x, H^{\bot}_0) = \inf_{y \in H^{\bot}_0} ||x-y||$

###### Ортогональное разложение
$H=H_0 \oplus H^{\bot}_0 \to x=x_{H_0}+x_{H^{\bot}_0}$
где $x \in H, \,\, x_{H_0} \in H_0, \,\, x_{H^{\bot}_0} \in H^{\bot}_0$
При этом расстояние до $H^{\bot}_0$ равно норме ортогональной проекции на $H_0$:
$p(x, H^{\bot}_0) = ||x_{H_0}||$

###### Проекция на одномерное подпространство
т.к. $H_0 = \{x_0\}$
$x_{H_0} = tx_0$,     $t$ нужно найти
Поскольку прямая задаётся вектором $x_0$​, это даёт уравнение
$(x-tx_0,x_0) = 0$
$(x,x_0) - t(x_0, x_0) = 0$
$(x,x_0) = t(x_0, x_0)$
$t=\frac{(x,x_0)}{(x_0, x_0)}$
$x_{H_0} = tx_0 = \frac{(x,x_0)}{(x_0, x_0)}x_0$
$x_{H_0} = \frac{(x,x_0)}{(x_0,x_0)}x_0$

###### вычислим норму этой проекции
$$||x_{H_0}|| = ||\frac{(x,x_0)}{x_0,x_0}x_0|| = \frac{|(x,x_0)|}{||x_0||^2} ||x_0|| = \frac{|(x,x_0)|}{||x_0||}$$
получили: $||x_{H_0}|| = \frac{|(x,x_0)|}{||x_0||}$

т.к. $p(x, H^{\bot}_0) = ||x_{H_0}||$  и $||x_{H_0}|| = \frac{|(x,x_0)|}{||x_0||}$  $\to$   $p(x, H^{\bot}_0) = \frac{|(x, x_0)|}{||x_0||}$
